# Memory Card 童趣化 UI 设计说明

## 设计目标

- 以儿童记忆游戏为场景，统一打造柔和、圆润、轻松的童趣视觉风格。
- 通过马卡龙配色、气泡面板、圆角按钮和轻量动效，降低儿童理解成本。
- 保持全场景一致的视觉语义，让启动、登录、主页、关卡、背包、设置、战斗和结算拥有同一套设计语言。

## 视觉系统

- 主背景：浅粉 `#FFF7FB` 到浅蓝 `#EEF9FF` 的柔和渐变。
- 主强调色：粉桃 `#FFB6CD`、奶油橙 `#FFC89A`、薄荷绿 `#95E4D4`、晴空蓝 `#A9D8FF`、薰衣草 `#D0C0FF`。
- 文本色：主文本 `#5C4D74`，辅助文本 `#816F97`。
- 字体：
  - 标题：`ZCOOL KuaiLe`
  - 正文：`Noto Sans SC` + `Nunito`
- 圆角体系：
  - 大面板：`32px`
  - 中型卡片：`24px`
  - 小型输入框 / 统计块：`16px - 20px`
- 阴影体系：使用低对比、偏粉紫的柔和阴影，避免硬边和强烈发光。

## 交互规范

- 所有主按钮使用气泡型胶囊轮廓，支持轻微弹跳反馈。
- 通知使用底部气泡提示，文案保持短句式、鼓励式表达。
- 加载状态采用“小动物跳动 + 星星进度条”组合，避免静态等待。
- 卡片翻面和按钮按压动画均支持“减少动态效果”开关。
- 设置面板提供增强对比度模式，方便在不同设备环境下使用。

## 场景映射

- 启动页：品牌标题 + 趣味加载动画。
- 登录 / 注册页：软萌说明文案 + 圆角表单。
- 主页大厅：欢迎信息、推荐关卡、快捷入口、陪伴角色提示。
- 关卡选择页：不同难度卡片 + 目标步数 + 奖励说明。
- 游戏主界面：顶部统计 + 左侧任务提示 + 中央卡片棋盘。
- 结算弹窗：星级评价、成绩摘要、奖励提示、再玩一次 / 回到大厅。
- 背包页：道具卡片化展示。
- 设置页：音乐、音效、减少动效、增强对比度切换。

## 资源说明

- 当前版本主要通过 CSS 变量、渐变、圆角和文本徽章完成视觉实现。
- 若后续需要导出更完整的切图包，可继续补充 SVG 云朵、彩虹、星星和动物插画资源。
- 当前 React 应用本身可作为可运行交互原型，用于验证页面切换、动效反馈和响应式布局表现。
